### Progress Props

| 属性       | 描述               | 类型                        | 可选值                                     | 默认      |
| ---------- | ------------------ | --------------------------- | ------------------------------------------ | --------- |
| **value**  | 进度条当前数值     | `number`                    | `-`                                        | `0`       |
| **max**    | 最大值             | `'number','string'`         | `-`                                        | `100`     |
| **colors** | 在范围内自定义颜色 | `{ [key: number]: string }` | `-`                                        | `{}`      |
| **type**   | 预定义的状体类型   | `'string'`                  | `'default', 'success', 'warning', 'error'` | `default` |
