### 固定颜色

指定一个默认颜色进行覆盖

:::playground

capacity/custom-color.vue

:::

### Capacity Props

| 属性      | 描述                | 类型                | 可选值 | 默认 |
| --------- | ------------------- | ------------------- | ------ | ---- |
| **value** | 当前的容量值        | `'string','number'` | `-`    | `0`  |
| **color** | 自定义容量 css 颜色 | `string`            | `-`    | `-`  |
