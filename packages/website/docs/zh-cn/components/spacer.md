### Spacer Props

| 属性       | 描述       | 类型                | 可选值 | 默认    |
| ---------- | ---------- | ------------------- | ------ | ------- |
| **x**      | 水平轴间距 | `'string','number'` | `-`    | `1`     |
| **y**      | 垂直轴间距 | `'string','number'` | `-`    | `1`     |
| **inline** | 行内模式   | `'boolean'`         | `-`    | `false` |
