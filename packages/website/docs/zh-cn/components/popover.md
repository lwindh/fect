# Popover / 气泡卡片

通过点击或鼠标移入触发的气泡风格弹出层

### 默认的

基础示例

:::playground

popover/default.vue

:::

### Popover Props

Props 和 <fe-link to="/zh-cn/components/tooltip">`ToolTip`</fe-link>一样,只是移除了`content`

### Propover Slots

| 插槽名   | 说明                            |
| -------- | ------------------------------- |
| `-`      | Popover 内嵌 `HTML` 文本        |
| `widget` | 触发 `Popover `显示的 HTML 元素 |
