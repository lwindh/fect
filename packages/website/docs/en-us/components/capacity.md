### Capacity Props

| Attribue  | Description      | Type                | Accepted values | Default |
| --------- | ---------------- | ------------------- | --------------- | ------- |
| **value** | value number     | `'string','number'` | `-`             | `0`     |
| **color** | css color string | `string`            | `-`             | `-`     |
