### Tag Props

| Attribue       | Description                 | Type                                       | Accepted values | Default   |
| -------------- | --------------------------- | ------------------------------------------ | --------------- | --------- |
| **text**       | tag content                 | `String`                                   | -               | `-`       |
| **type**       | tag type                    | `'default', 'success', 'warning', 'error'` | -               | `default` |
| **use-invert** | invert color and background | `boolean`                                  | -               | `false`   |
