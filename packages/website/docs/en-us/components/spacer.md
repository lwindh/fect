### Spacer Props

| Attribue   | Description    | Type                | Accepted values | Default |
| ---------- | -------------- | ------------------- | --------------- | ------- |
| **x**      | x-axis spacing | `'string','number'` | `-`             | `1`     |
| **y**      | y-axis spacing | `'string','number'` | `-`             | `1`     |
| **inline** | inline mode    | `'boolean'`         | `-`             | `false` |
