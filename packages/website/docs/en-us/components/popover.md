# Popover

The floating box popped by clicking or hovering.

### Default

:::playground

popover/default.vue

:::

### Popover Props

Props same as <fe-link to="/en-us/components/tooltip">`ToolTip`</fe-link>,only remove `content`

### Propover Slots

| Slot name | Description                          |
| --------- | ------------------------------------ |
| `-`       | `-`                                  |
| `widget`  | `HTML element that triggers popover` |
